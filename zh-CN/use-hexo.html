use-hexo

之前一直想在github弄个blog，用了一下jekyll，选了好几个模版，就想着自己改个模版出来，还要弄分页什么的。最后由于不熟悉前端的东西，就从入门到放弃。

今天终于又想起这件事了，决定试用hexo。于是，搞定啦。。。

hexo的文档比较清楚好理解，坑少。windows下按着操作完全ok。

然后。。。我打算在自己的Mac上装就出了点问题。

Mac上安装hexo的时候，要用 npm install hexo-cli -g –no-optional。
在这之前，还把brew升级了一下。

用默认的模版做了点小的改动，我的blog就搞定啦。

终于完成在github上部署blog。开心。
